# TV Player for VIDAA

Статическая сборка персонального IPTV-плеера для VIDAA.

- Приложение запускается напрямую из GitHub Pages.
- Адреса IPTV и медиатеки не записаны в репозиторий: они загружаются из удалённого конфига.
- Каталог `epg/` содержит подготовленные небольшие фрагменты программы передач.

Публичный URL приложения после включения GitHub Pages:

`https://<github-user>.github.io/tv-player-vidaa/`
